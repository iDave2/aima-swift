EuclideanEnvironment

public protocol EuclideanEnvironment

A protocol for Euclidean environments used in Chapter 2 of AIMA3e, this represents the world as a grid of Cartesian coordinates, a Space, that agents move around in.

Remember: These protocols are not polymorphic like classes; review this and test, test, test.

  • scores

    Cumulative scores from each judge (performance measure) for each agent.

    At each time step, each agent in the environment attempts to perform an action and each judge scores the effort by observing actual changes to the environment. This dictionary of dictionaries keeps track of these associations and scores.
